## 2. 核心概念

1. `Agent` 产品不是单独的模型，也不是单独的流程编排，而是 `Model + Harness`。
2. `Agency` 来自模型训练；本仓库重点学习的是如何构建 `Harness`。
3. `Harness` 由工具、知识、观察、行动接口和权限边界组成。
4. 本仓库主线是 `coding agent harness`：从最小 `agent loop` 开始，逐步叠加工具调用、权限控制、`hooks`、上下文压缩、记忆、后台任务、团队协作、`worktree` 隔离和 `MCP`。
5. 仓库同时保留新旧两条教程轨道：根目录 `s01_*` 到 `s20_*` 是新的 `20` 课主线；`docs/`、`agents/` 和 `web/` 中还有旧的 `12` 课过渡内容。

# 每日学习模板

建议每天投入 `1.5` 到 `2.5` 小时。

## 时间分配

```text
20 分钟：读 README / docs
40 分钟：读 code.py
30 分钟：跑代码或测试
20 分钟：手动改一个小点
10 分钟：写当天总结
```

## 每日检查清单

每天学习结束前，回答下面 5 个问题：

1. 今天新增的核心能力是什么？
2. 它解决了 `agent harness` 中的哪个工程问题？
3. 它依赖前面哪些模块？
4. 如果没有这个模块，会出现什么失败模式？
5. 我能否用自己的话重新实现一个简化版本？

## 复盘标准

当天学习不以“看完”为完成标准，而以能否说清楚下面三件事为标准：

1. 这个模块输入什么。
2. 这个模块输出什么。
3. 这个模块改变了 `agent harness` 的哪一部分能力。
